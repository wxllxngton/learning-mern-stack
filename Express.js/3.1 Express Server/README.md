### Running our app

#### Initialize npm without questions

```bash
npm init -y
```

#### Installing Express.js

```bash
npm i express
```

#### Changing module type to ECMA Scripts

```json
    "main": "index.js",
    // Add this line
    "type": "module",
```

#### Importing the modules

```javascript
import express from 'express';
```

#### Running script

```bash
node index.js
```
